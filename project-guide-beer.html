Cheers to the guides who have lead us through rapid waters, tall trees, and the uncharted terrain of the world and ourselves.

Sweetwater Brewery made a beer specifically to honour trusted guides. Using the designs created by 22squared, I built the Guide Beer website dedicated to the tribute brew. Each can of Guide Beer features a map of a location where the guides worked. Built on React.js with a bunch of CSS animation, this website is dynamic as well as beautiful.
